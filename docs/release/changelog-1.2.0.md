# Happy Creeper 1.2.0

First stable release for NeoForge on Minecraft 1.21.7.

## Added

- Tamed creeper ability system
- `Lava Biscuit` permanently grants the Fire Resistance ability
- `Fish Biscuit` permanently grants the Swim Speed ability
- `Extreme Blast Biscuit` permanently grants the Extreme Blast ability (twice as powerful combat blasts)
- `Slime Biscuit` permanently grants the Slime Jump ability (leap attacks with sticky landing on enemies)
- Active abilities displayed in the companion UI
- Already-granted abilities cannot be applied a second time

## Fixed

- Tamed creeper AI no longer jitters during combat from navigation being overridden by the ability handler

## Notes

- First stable release — no longer a beta build
- All features from 1.2.0-beta.1 are included and stable
- Existing worlds from 1.2.0-beta.1 should continue to work
- Ability data is stored per-creeper and persists across sessions
- This file targets Minecraft `1.21.7` on NeoForge
