# Happy Creeper 1.0.0-alpha.1

First public alpha release.

## Added

- Full creeper weakening and taming loop
- Biscuit, Anti-Blast Biscuit, Sweet Gunpowder Biscuit, and Rainbow Biscuit
- Creeper-head protection for safe approach
- Weakened creeper state with visual feedback and failed explosion behavior
- Tamed creeper owner system
- Follow and stay commands
- Teleport catch-up when the creeper falls behind
- Cross-dimension follow support
- Respawn resync for owned creepers
- Companion combat with contact blast / shove attacks
- Larger non-destructive death burst for tamed creepers
- Healing with Gunpowder and Sweet Gunpowder Biscuit
- Tamed creeper recolors
- Rainbow creeper mode
- Rainbow Biscuit chest loot

## Notes

- This is an alpha build meant for public playtesting
- Balance and visuals may change in later versions
- Existing worlds should generate Rainbow Biscuit loot only in newly generated structure chests
