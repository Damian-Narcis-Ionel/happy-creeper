# Happy Creeper

Short summary:

Turn creepers into survival companions by weakening, taming, healing, recoloring, gearing them up, and fighting alongside them.

## Overview

Happy Creeper is a NeoForge mod for Minecraft 1.21.5 that gives creepers a full companion gameplay loop.

Instead of only running from them, you can approach them safely with a creeper head, weaken them with a special biscuit, tame them, and keep them with you as a combat companion.

This mod is built around survival-friendly mechanics rather than creative-only gimmicks.

## How It Works

1. Wear a Creeper Head or Happy Creeper Mask to get close safely.
2. Feed the creeper an Anti-Blast Biscuit to weaken it.
3. Feed the weakened creeper a Sweet Gunpowder Biscuit to tame it.
4. Right-click your tamed creeper with an empty hand to open its companion UI.
5. Use Shift + right-click with an empty hand to switch between follow and stay in the world.
6. Heal it directly or let it use stored fuel automatically when hurt.

## Tamed Creeper Features

- Follows its owner and can be told to stay
- Teleports back if it gets left behind
- Changes dimension with its owner
- Resyncs after the owner respawns
- Works correctly with leads
- No longer counts as a nearby monster for sleeping
- Defends the owner from hostile mobs
- Attacks mobs the owner is fighting
- Keeps fighting mobs that switch aggro onto the creeper itself
- Uses a small contact blast attack instead of normal self-destruction
- Dies with a larger non-destructive final explosion
- Has increased health compared to a normal creeper
- Can equip helmets and chestplates
- Can store fuel and automatically self-heal below half health

## Visual Features

- Weakened creepers show a continuous visual effect
- Tamed creepers use a custom happy green texture by default
- Tamed creepers can be recolored with dyes
- Reusing the same dye does not waste the item
- Rainbow Biscuit unlocks a smooth rainbow mode that cycles through colored creeper textures

## Companion UI

- 3D creeper preview
- Helmet slot
- Chestplate slot
- Fuel slot
- Health display
- Active status effect icons
- Follow/stay button
- Shows the creeper's custom name when named with a name tag

## Items

- Biscuit
- Anti-Blast Biscuit
- Sweet Gunpowder Biscuit
- Rainbow Biscuit
- Happy Creeper Mask

## Current Release Notes

This release is intended as a wider beta playtest build for Minecraft 1.21.5.

The taming loop is fully playable, with companion armor management, potion support, recolors, rainbow mode, a dedicated creeper UI, and fuel-based self-healing in place. Balance and polish are still in progress.
